# Camila Block - Jogo Web

Um jogo de puzzle de blocos intenso e romântico, feito com carinho para a Camila.

## 🎮 Como Jogar

1. **Escolha uma peça** da base da tela
2. **Toque no tabuleiro** para encaixar a peça
3. **Limpe linhas e colunas** completas para ganhar pontos
4. **Sobreviva** enquanto a dificuldade aumenta

## 🌟 Características

- 🎯 Lógica de puzzle desafiadora que começa difícil
- 💗 Mensagens românticas durante o jogo
- 📊 Sistema de pontuação e níveis progressivos
- 💾 Salva seu progresso automaticamente
- 🏆 Recordes locais persistidos
- 📱 Totalmente responsivo para celular e desktop
- ✨ Interface bonita com gradientes e efeitos

## 🚀 Como Usar

### Localmente

1. Clone ou baixe este repositório
2. Abra o arquivo `index.html` no seu navegador
3. Comece a jogar!

### Online

Hospede os arquivos em qualquer servidor web estático:
- GitHub Pages
- Vercel
- Netlify
- Ou qualquer outro host

## 💾 Armazenamento Local

O jogo salva automaticamente:
- Sua partida em andamento
- Seus recordes (top 10)
- Melhor pontuação
- Nível máximo alcançado

Tudo é armazenado no `localStorage` do navegador.
